Start an instant CRUD interface for any Postgres or MySQL database.

Use it via the CLI:

$ npm install -g helppo-cli
$ helppo-cli <connection_string>

Or mount it in your NodeJS express application:

import helppo from 'helppo';
app.use(helppo());
